Overview


For Hacktoberfest 2017 my roommate and I both decided to take on a little project. I'm all about a free T-Shirt and a challenge so I set about making a game. My experience with Python was (is) still somewhat new so I had to limit my goals to something attainable in a month.

    Goals:
  • Create a functional Hangman Game
  • Successfully use an API to select a random word
  • Visually display letters/hangman images
  • Practice the GitHub/Git flow

I ended up with a half decent console application and was pretty pleased with myself. It was the first game I coded so I'm still pretty proud of it. Looking back even now at the code I can't help but critique it but I like to leave it as a reminder. As long as my Hacktoberfest projects each year improve in complexity then I'm satisfied.
